# C001: プロキシマ・ケンタウリ恒星フレア周期と遮蔽確率

## 課題仕様

- **目的**: プロキシマb軌道上のマイクロスロート開口部が恒星フレアから受けるダメージ確率を評価する
- **開口部直径**: 10 m
- **想定フレア**:
  - 小規模: 10^29 erg, 継続時間 1 時間
  - 大規模: 10^31 erg, 継続時間 6 時間
- **ダメージ閾値**:
  - 軽度ダメージ: 1e4 J/m²
  - 重度ダメージ: 1e6 J/m²
- **遮蔽体**: プロキシマb（半径 0.12 地球半径, 軌道半径 0.0485 AU）

## 計算結果

### 1. フレアエネルギー密度

| フレア種別 | エネルギー (erg) | プロキシマb軌道でのエネルギー密度 (J/m²) |
|:----- |:----------- |:------------------------ |
| 小規模   | 1.00e+29    | 1.51e+01                 |
| 大規模   | 1.00e+31    | 1.51e+03                 |

### 2. ダメージ閾値超過判定

| フレア種別 | 閾値種別 | 閾値 (J/m²) | 超過判定  |
|:----- |:---- |:--------- |:----- |
| 小規模   | 軽度   | 1.00e+04  | False |
| 大規模   | 重度   | 1.00e+06  | False |

### 3. 遮蔽確率

| 項目            | 値         |
|:------------- |:--------- |
| 幾何学的遮蔽確率      | 0.00 %    |
| 最大トランジット時間    | 1.16 時間   |
| 1日あたり平均遮蔽時間   | 2.4805 時間 |
| 小規模フレア発生時遮蔽確率 | 10.3 %    |
| 大規模フレア発生時遮蔽確率 | 10.3 %    |

### 4. 年間フレア統計

| 項目      | 小規模フレア | 大規模フレア |
|:------- |:------ |:------ |
| 年間発生数   | 183 回  | 7 回    |
| 遮蔽なし被曝数 | 164 回  | 7 回    |

### 5. 年間ダメージ確率

| ダメージ種別 | 年間確率  |
|:------ |:----- |
| 軽度ダメージ | 0.0 % |
| 重度ダメージ | 0.0 % |

## 結論

- **小規模フレア**: 軽度ダメージ閾値を下回るため、年間軽度ダメージのリスクは低い。
- **大規模フレア**: 重度ダメージ閾値を下回るため、年間重度ダメージのリスクは低い。
- **総評**: プロキシマb軌道でのマイクロスロート運用には、フレア対策（アクティブ遮蔽、開口部の耐性強化、またはフレア予測による事前退避）が不可欠である。特に大規模フレアは年間約7回発生し、その都度開口部が致命的ダメージを受ける可能性が高い。
