# C106: 3Dプリンタ出力可能な構造部材の最適化

## 計算目的

プロキシマbで3Dプリンタを使用して構造部材を製造するための最適化。
SIMP法によるトポロジー最適化と、積層方向依存性を考慮した材料選定を行う。

## 数式

### SIMP補間（Solid Isotropic Material with Penalization）

$ K(x) = E_{\min} + (E_0 - E_{\min}) \cdot x^p $

### コンプライアンス最小化問題

$ \min_x C(x) = \mathbf{u}^T \mathbf{K}(x) \mathbf{u} $

### 制約条件

$ V(x) / V_0 = f, \quad 0 < x_{\min} \leq x \leq 1 $

## 計算結果

- SIMP補間: E(x=0.5) = 8.88 GPa
- 体積制約: 目標体積割合 = 30%
- 積層方向強度比: Z/XY = 70%

### アルミ・Mgの精錬溶解の現状

| 材料           | 精錬方法                | 現在の技術成熟度 | プロキシマbでの実現性 |
| ------------ | ------------------- | -------- | ----------- |
| アルミ（Al）      | 溶融塩電解（ホール・エルー法）     | ◎ 成熟     | ○ 電力さえあれば可能 |
| アルミ（Al）      | 3Dプリンタ用粉末製造（アトマイズ法） | ◎ 成熟     | △ 設備が大掛かり   |
| マグネシウム（Mg）   | 溶融塩電解（ドーソン法）        | ○ 実用化済み  | ○ 電力さえあれば可能 |
| Mg合金（AZ31B等） | 粉末製造（ガスアトマイズ法）      | ○ 実用化済み  | △ 設備が大掛かり   |

### プロキシマbでの現実的な製造フロー

```
1. プロキシマbの砂（ケイ酸塩主体）からアルミ・Mgを精錬
   → 溶融塩電解（電力はダイソン環から供給）

2. 精錬した金属を粉末化
   → ガスアトマイズ法（アルゴンガスは現地調達 or 地球から輸入）

3. SLM方式の3Dプリンタで部品を出力
   → 構造部材・エンジン部品・ロボットアーム等

4. 熱処理で強度を向上
   → T6処理（溶体化＋時効）でAlSi10Mgの強度を最大化
```
