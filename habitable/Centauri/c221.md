## 量子通信 vs 光通信：比較と結論

### 結論：**量子通信は「絶対的な安全性」と「量子状態の転送」で光通信を上回るが、速度・帯域・実用性では光通信が優れる。用途によって使い分けるべき。**

---

### 量子通信と光通信の比較

| 項目    | 内容                         |
| ----- | -------------------------- |
| 冷却方式  | パルス管冷凍機（機械式、メンテナンスフリー）     |
| 冷却温度  | 9K（量子コンピュータ動作温度）           |
| 消費電力  | 1台あたり 約100W                |
| サイズ   | 10cm×10cm×20cm（各ロボットに搭載可能） |
| 台数    | 10万台（全自己複製ロボット）            |
| 総消費電力 | 10MW（ダイソン環出力の約0.6%）        |

### ケンタ時代の使い分け

| 用途           | 通信方式      | 理由                |
| ------------ | --------- | ----------------- |
| AR空間の映像転送    | 光通信（従来）   | 帯域が必要。盗聴されても問題ない。 |
| 重要データ（設計図）   | 光通信（暗号化）  | 帯域が必要。暗号化で十分安全。   |
| 量子コンピュータ間の同期 | 量子通信      | 量子状態を転送する必要がある。   |
| 認証・署名        | 量子通信      | 絶対的な安全性が必要。       |
| マイクロスロート経由   | 量子通信（理論上） | 0秒通信＋量子状態転送。      |
