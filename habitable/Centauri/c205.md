### 人工ダイヤモンド製造



| 項目   | 内容                          |
| ---- | --------------------------- |
| 製造方法 | CVD（化学気相成長法）または爆轟法          |
| 原料   | プロキシマbの大気中のCO2または地下の炭素源     |
| 製造場所 | プロキシマb地下基地内の専用工場            |
| 生産能力 | 初期: 1kg/日、拡張後: 100kg/日      |
| 用途   | NVセンター用ナノダイヤモンド、放熱用ダイヤモンド基板 |



### 冷却装置の実装

| 項目    | 内容                         |
| ----- | -------------------------- |
| 冷却方式  | パルス管冷凍機（機械式、メンテナンスフリー）     |
| 冷却温度  | 9K（量子コンピュータ動作温度）           |
| 消費電力  | 1台あたり 約100W                |
| サイズ   | 10cm×10cm×20cm（各ロボットに搭載可能） |
| 台数    | 10万台（全自己複製ロボット）            |
| 総消費電力 | 10MW（ダイソン環出力の約0.6%）        |
