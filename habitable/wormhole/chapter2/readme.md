# 課題ID #3011：プロキシマ側受電拠点配置（Scattering Algorithm）

## 1. 課題概要

21年の航行（#1023）を終え、目的地に到達した「865拠点パッケージ」を、ブラックホールの事象の地平面わずか外側である**最内安定円軌道（ISCO: 1.1818 M）**へ均等配置する 1, 2。この配置は、将来的な地平面ダイソン環（#1008）の基礎グリッドとなり、地球からの1,200TWレーザー受電およびペンローズ過程によるエネルギー回収効率を最大化することを目的とする 。

## 2. 数理モデル：フィボナッチ螺旋による均等散布

865個の独立した拠点を球面上に均等配置するため、金星テラフォーミング（#102a）で最適化されたフィボナッチ螺旋アルゴリズムを採用する 。

配置座標 $(x, y, z)$ は以下の通り定義される：$ z_i = \left( 1 - \frac{1}{N} \right) \left( 1 - \frac{2i}{N-1} \right) $

 $r_i = \sqrt{1 - z_i^2} $

$ \theta_i = \pi (1 + \sqrt{5}) i $

$ x_i = r_i \cdot \cos(\theta_i) \cdot R_{ISCO}, \quad y_i = r_i \cdot \sin(\theta_i) \cdot R_{ISCO}, \quad z_i = z_i \cdot R_{ISCO} $

$ここで $N=865、$R_{ISCO} = 1.1818 M$ である 。
